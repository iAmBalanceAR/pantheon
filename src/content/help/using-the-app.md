## Top navigation

- **Projects** — List of all projects, grouped by status (in progress, future, complete).
- **Settings** — Platform-level options: keys, defaults, **Agent Roster**, controller profiles if enabled.
- **New** — Shortcut to create a project.
- **Help** — This learning center (guides, troubleshooting, FAQ).

## Project list

Each row shows status, tier (when relevant), and date. Hover actions may include pause, resume, or delete depending on state. **Deleting** a project removes its data from Pantheon subject to your backend rules.

## Inside a project

The **header** shows the project name and tabs. A **pause / resume** control appears when the project can be toggled. Use pause when you need to stop all agent execution safely.

## Real-time updates

While a project runs, **Overview**, **Chat**, **Team**, and related views may update **live** as database rows change. If something looks stuck, refresh once — but usually the connection picks up new messages and status automatically.

## Settings vs project settings

| Location | What it affects |
|----------|------------------|
| **Settings** (top nav) | Your account preferences, agent custom context, API keys |
| **Project → Settings** | That project's name, budgets, per-agent model overrides |

Changes to **Agent Roster** apply to **future task runs** for all projects under your account (they are read when each task executes).
